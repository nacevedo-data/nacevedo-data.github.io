Análisis del Sistema de Subte de Buenos Aires

Python · MySQL · Power BI · Data Warehouse

Proyecto de análisis end-to-end sobre el sistema de transporte subterráneo de Buenos Aires. A partir de datos de 83 millones de registros (2014–2021), construí un pipeline completo que va desde la exploración en Python hasta un dashboard interactivo en Power BI, pasando por un data warehouse en MySQL con arquitectura estrella.

La narrativa central del proyecto: "El Subte es un sistema en tensión: alta demanda, tarifas crecientes y una accesibilidad que no acompaña el crecimiento."
